技术背景

在许久之前写的一篇博客中，我们介绍过使用twine向pypi上传我们自己的开源包的方法。最近发现这个方法已经不再支持了（报错信息如下所示），现在最新版需要使用API Token进行文件上传，这里大致介绍一下配置的方法。
