技术背景

PySAGES是一款可以使用GPU加速的增强采样插件，它可以直接对接到OpenMM上进行增强采样分子动力学模拟，这里我们测试一下相关的安装，并尝试跑一个简单的增强采样示例。

安装PySAGES

PySAGES本身可以使用pip进行安装：

python3 -m pip install git+https://github.com/SSAGESLabs/PySAGES.git
